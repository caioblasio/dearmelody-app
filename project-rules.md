Use:
- React + TypeScript
- Tailwind
- React Router SPA for client side routing
- shadcn/ui

Rules:
- only build the frontend, the backend is a separate service
- For APIs you can use a mock endpoint with dummy data
- Create a reusable helper for API calls
- You are free to assume the API contract based on the UI
- prefer composition over abstraction
- keep components small
- avoid premature optimization
- use TanStack Query for server state
- forms use RHF + Zod
